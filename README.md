# 🤖 AI FAQ Chatbot with GUI

## 📌 Project Overview

The AI FAQ Chatbot with GUI is a Python-based application that automatically answers frequently asked questions using Natural Language Processing techniques. The chatbot compares user queries with a dataset of predefined questions and returns the most relevant response.

A graphical user interface is built using Tkinter, allowing users to interact with the chatbot in a simple chat window.

The chatbot uses TF-IDF vectorization and cosine similarity to determine the best matching response.



## 🎯 Features

- Interactive Graphical User Interface  
- Chat-style conversation window  
- Automatic response generation  
- Scrollable chat history  
- Send message using Enter key  
- Lightweight and beginner-friendly project  



## 🧠 Technologies Used

- Python  
- Tkinter (GUI)  
- Pandas (dataset handling)  
- Scikit-learn (TF-IDF and similarity)  
- Natural Language Processing (NLP)

## ⚙️ Installation

Clone the repository:
git clone https://github.com/yourusername/ai-chatbot-project.git

Move into the project folder:
cd ai-chatbot-project


Install required libraries:
pip install pandas scikit-learn



## ▶️ Run the Application

Run the chatbot using:
python chatbot_ui.py

A chat window will open, allowing you to interact with the chatbot.
